# RFill Tools

RFill Tools is a Blender addon that adds procedural geometric fill tools to the painting workflow.  
The addon allows artists to draw filled shapes directly inside Blender using simple click-and-drag interactions.

Current features include a rectangle fill tool under Image Editor> Paint> Left side toolbar.

RFill Tools focuses on responsive viewport interaction, clean UX, and scalable shape-based painting systems.  
Future updates will expand the toolkit with additional geometric fill primitives.

This project is based on and inspired by the excellent Atelier Sculpt addon by JF Matheu: [Atelier Sculpt GitHub Repository](https://github.com/jfranmatheu/Atelier-Sculpt).
